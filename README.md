
## TaskFlow

Gerenciador de tarefas Fullstack com **Laravel 11** e **React (Vite)**. Projeto desenvolvido para atender a todos os requisitos de CRUD, persistência e diferenciais técnicos.

## Tech Stack

* **Backend:** PHP 8.2 | Laravel 11 (API REST)
* **Frontend:** React | TypeScript | Tailwind CSS | Shadcn/UI
* **Banco:** MySQL / PostgreSQL
* **Extras:** Driver.js (Tour), SWR (Cache), Lucide Icons

---

## Funcionalidades

* **CRUD Completo:** Criar, listar, editar e deletar tarefas.
* **Filtros:** Busca por texto, prioridade e ordenação inteligente.
* **Autenticação:** Sistema de login seguro para múltiplos usuários.
* **Lixeira:** Recuperação de tarefas (Soft Delete).
* **Onboarding:** Guia interativo para novos usuários.
